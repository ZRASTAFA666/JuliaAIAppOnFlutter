# Installation Guide

## Prerequisites

- Flutter SDK
- Julia (v1.9+ recommended)
- Node.js (for web support)

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/sorydima/JuliaAIAppOnFlutter.git
   cd JuliaAIAppOnFlutter
   ```

2. Get Flutter dependencies:
   ```bash
   flutter pub get
   ```
